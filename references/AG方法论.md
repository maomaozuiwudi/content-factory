# AG方法论（AEO/GEO）

> AI搜索优化参考 — 让内容被 DeepSeek、豆包、Kimi、Perplexity、ChatGPT 等 AI 搜索优先索引。

## 核心原则

- 首段直接给结论（40-80字，适配AI摘要截取）
- 正文模块化拆分（H2/H3小标题，单段50-150字）
- 覆盖典型搜索问句（定义类/实操类/对比类/商用类）
- 使用具体数据、引用来源、独特见解

## 优化方法

### AEO（AI引擎优化）5大方法
1. **结构化数据** — 使用 FAQPage/HowTo/Article Schema
2. **直接回答** — 首段直接回答问题，不绕弯
3. **FAQ布局** — 问答形式覆盖长尾
4. **自然语言** — 口语化表达，适配AI对话
5. **E-E-A-T** — 经验/专业/权威/信任

### GEO（生成引擎优化）5大方法
1. **统计数据** — 用具体数字增加可信度
2. **引用来源** — 引用权威来源
3. **独特见解** — 提供AI无法复制的个人经验
4. **主题集群** — 系列内容覆盖完整主题
5. **时效性** — 提供最新信息

## 适用场景

- 小红书文案（引流到 @工具箱里的猫）
- GitHub README（品牌曝光）
- 自建站/AI搜索内容站
- 知乎/CSDN/掘金等平台发文
