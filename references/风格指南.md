# 风格指南

> 把你看过、写过、收藏过的喜欢的小红书文案贴进来，系统会自动学习你的风格。
> 贴得越多，出文越准。

## 可以放什么？

- 你之前发过的点赞高的帖子
- 你喜欢的博主的文案（注明是谁的风格）
- 你觉得写得好的段落
- 你对调性的描述（如：直白实用风/文艺感性风/东北唠嗑风）

## 格式参考

```
来源：@某某博主
风格：直白实用、带数字、短句多
例句：
「这3个工具我用了半年，第2个真的绝了」
「姐妹们别踩坑了，这个真的不行」
```
