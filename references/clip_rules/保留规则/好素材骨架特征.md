# 保留规则 — 好素材骨架特征

基于5个已剪辑视频的骨架分析（2,641帧，轮廓系数0.93），提炼出的高复现姿态特征。
用于判断哪些姿态是"好素材"，应予以保留。

---

## 核心姿态：直立口播 / 半身演示

**出现频率：99.5%**（5个视频全部共享）

**特征描述：**
- 人物直立，居中构图
- 手臂微屈，手下垂在腰部附近
- 双腿直立（半身出镜时可能不可见）
- 肩部对称（左肩在中心偏右，右肩在中心偏左 → 人物正对镜头）

**关键参数范围（归一化坐标，以双髋中心为原点、肩宽为缩放单位）：**

**关节角度：**
- 肩-肘-腕平均角度：152.3°
- 膝盖弯曲度：177.9°（接近挺直）
- 手部高度（相对头部）：1.97（手在腰部，低于头）

**好素材自动判定规则：**

对任意视频帧，归一化骨架与以上姿态的欧氏距离 < 1.98（约3倍标准差）→ 判定为"好素材姿态"。

## 剪辑建议

基于骨架分析，针对口播/演示类视频的建议：

1. **主体结构**：人物保持站立/半身居中出镜，这是最稳定的主体镜头
2. **手部位置**：手臂微屈时手在腰部高度（归一化y约1.5-2.0），是最自然的表述姿势
3. **镜头构图**：肩部保持对称（左肩x正、右肩x负），人物正对镜头时最佳
4. **过渡帧处理**：跳出标准姿态范围的零星帧（骨架不完整/跳跃/静止）可剪掉，剩余的主体段自动保留
5. **时长分配**：若需合成15秒成片，3个视频各取5秒最连贯段拼接
