# 火柴人骨架剪辑规则

基于 MediaPipe PoseLandmarker 的智能视频剪辑，4条核心规则。

---

## Rule 1 — 骨架扭曲检测（剪掉）

**不剪的正常动作：**
- ✅ 正常弯手（摸脸、挠头、挥手、招手）
- ✅ 转身/侧身（单侧关键点部分不可见）
- ✅ 抬腿/弯腿（走路、跑步、蹲下、跳跃）
- ✅ 手臂自然下垂（三点共线）

**才剪的骨架异常：**
- ❌ 躯干关键点（鼻/双肩/双髋）一帧内跳跃超过身体高度的40%，且5点中3点以上同时跳跃 → 骨架数据飞了
- ❌ 双手同时举到肩膀以上且肘部弯到极限角度（<20°）→ 蜷缩姿态

**实现：** 逐帧帧间位移检测为主，极端蜷缩为辅。

---

## Rule 2 — 骨架不完整检测（剪掉）

**先判断全身/半身：**
- 扫描全视频，统计脚踝（27,28号关键点）可见帧占比
- 可见 > 30% → 全身，否则 → 半身

**判断标准：**
- 全身：需要鼻(0) + 双肩(11,12) + 双髋(23,24) + 至少一脚踝(27/28)
- 半身：需要鼻(0) + 双肩(11,12) + 至少一手腕(15/16)

**缺失期望关键点 → 剪掉。**

---

## Rule 3 — 静止检测（剪掉）

- 鼻子(0) + 双腕(15,16) 帧间位移 < 5像素
- 持续时间 > 1秒（30帧@30fps）
- → 判定为静止，剪掉

**注意：** 正常动作中偶尔的卡顿（<1秒）不算静止。

---

## Rule 4 — 选最连贯片段

1. 排除 Rule 1-3 标记的坏帧
2. 剩下的好帧中找 **最长的连续好段**
3. 从该段中间截取目标时长（自动避开头尾入画/出画）
4. **时长分配：**
   - N个视频，合成T秒 → 每个保留 T/N 秒
   - 例：3个视频剪成15秒 → 每个取5秒
   - 档位：3s / 5s / 8s

---

## 参数表

| 参数 | 默认值 | 说明 |
|:----|:------|:-----|
| jump_threshold | 0.4 (40%) | 躯干点帧间位移超过身体高度此比例 → 数据异常 |
| fold_angle | 20° | 肘部弯到低于此角度且双手同时 → 蜷缩 |
| stillness_threshold | 5px | 帧间位移小于此值 → 判定静止 |
| stillness_max | 1.0s | 连续静止超过此秒数 → 剪掉 |
| body_type_ratio | 0.3 | 脚踝可见帧占比超过此 → 全身 |
| confidence | 0.5 | MediaPipe 关键点置信度阈值 |
