# 配图偏好

> 告诉系统你喜欢什么风格的配图。
> 包括配色、构图、排版习惯。

## 可以放什么？

- 你常用的配色方案（如：深色科技风/莫兰迪色系/高饱和度）
- 你喜欢的排版风格（大字报/简约留白/拼图/对比图）
- 参考图描述或链接
- 不喜欢什么风格（红色背景别用/不要卡通字体等）

## 格式参考

```
常用配色：#1A1A2E 深蓝底 + #E94560 亮红点缀
字体偏好：粗体无衬线，标题64px+，正文36px+
排版：封面大字+3张功能卡片+1张对比图
不要：卡通风格、浅色背景、手写体
```
