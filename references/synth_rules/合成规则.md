# 合成规则记忆库

> 记录每次合成操作的参数偏好，越用越懂用户的合成习惯。

## 当前已知规则

### 基础参数
- 默认分辨率: 1080×1920 (竖屏)
- 默认帧率: 30fps
- BGM混音音量: 15%
- 配音音量: 100%

### 纯穿衣展示（模式一）
- 素材类型询问 → 选择 1️⃣ 纯穿衣展示
- 骨架剪辑合成（剔出+保留+最佳段选）
- 分段时长由总时长均分

### 带产品展示（模式二）
- 素材类型询问 → 选择 2️⃣ 带产品展示
- 用户分组：穿衣展示 / 产品展示（按序号）
- 每对时长分配: **1:4**（产品展示1 : 穿衣展示4）
- 产品视频: ffmpeg 固定时长裁剪（不做骨架，从中间取段）
- 穿衣视频: ClipVideo 骨架剪辑（剔出+保留+最佳段选）
- 拼接顺序: 产品a → 穿衣a → 产品b → 穿衣b ...

### 合成顺序规则

- **骨架大小排序** — 合成时按每段视频中人物骨架的大小（bbox面积）从小到大排列
- **目的**：防止远小近大切分镜时视觉跳跃
- **实现方式**：ClipVideo 骨架扫描时记录每帧的骨架框面积（bbox），取平均值作为该段视频的骨架大小分数
- **例外**：用户手动指定顺序时，以用户指定为准
